## Requirements

- Nix with flakes enabled
- Proxmox VMs (or any NixOS‑compatible host with EFI boot)
- SSH access to targets

**Proxmox:** Enable QEMU Guest Agent:

```bash
qm set <vmid> --agent 1        # Options -> QEMU Guest Agent
qm set <vmid> --serial0 socket # Hardware -> Serial port
```

## Quick Start

```bash
# 1. Generate admin SSH key
ssh-keygen -t ed25519 -f ~/.ssh/k3s-admin -N ""
cp ~/.ssh/k3s-admin.pub keys/admin.pub

# 2. Generate cluster secrets (WireGuard keys + K3s token)
nix run .#secrets --
git add -A && git commit -m "secrets"

# 3. Deploy the init node
nix run .#deploy -- init master1 -i ~/.ssh/k3s-admin

# 4. Wait for K3s, then join the remaining nodes
ssh root@<master1-ip> kubectl get nodes
nix run .#deploy -- rebuild master2 -i ~/.ssh/k3s-admin
nix run .#deploy -- rebuild master3 -i ~/.ssh/k3s-admin

# 5. Fetch kubeconfig (replaces WG IP with 127.0.0.1)
nix run .#get-kubeconfig -- master1 ~/.ssh/k3s-admin
```

## Commands

| Command                                                     | Description                                |
| ----------------------------------------------------------- | ------------------------------------------ |
| `nix run .#secrets --`                                      | Generate WireGuard keys + K3s token        |
| `nix run .#deploy -- init <node> [key] [--jump bastion]`    | Initial deployment (nixos‑anywhere)        |
| `nix run .#deploy -- rebuild <node> [key] [--jump bastion]` | Rebuild an existing node                   |
| `nix run .#deploy -- all [key] [--parallel]`                | Rebuild all nodes                          |
| `nix run .#deploy -- rollback <node> [key]`                 | Rollback to previous generation            |
| `nix run .#get-kubeconfig -- <node> [key]`                  | Fetch kubeconfig and point it to localhost |
| `nix run .#image -- add <file> [node\|all] [key]`           | Import a container image                   |
| `nix run .#image -- list [node\|all] [key]`                 | List images on nodes                       |
| `nix run .#image -- rm <ref> [node\|all] [key]`             | Remove an image                            |
